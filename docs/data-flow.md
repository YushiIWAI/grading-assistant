# データフロー図（法務相談用）

## サービスの概要

教員向けの記述式採点支援ツール。Google Classroomの回答データ（CSV）を取り込み、AIが仮採点と生徒向けフィードバックを生成する。最終判断は教員が行う。

## 法務確認のポイント

### Q1: 匿名化済みデータは「個人情報」に該当するか？

Gemini APIに送信されるのは:
- 仮番号（S001, S002...） — 個人を特定できない
- 回答テキスト — 通常は個人を特定する情報を含まない
- 採点基準 — 個人情報ではない

**主張**: 匿名化処理により、API送信データ単体では特定の個人を識別できないため、個人情報保護法上の「個人情報」に該当しない。したがって、外国にある第三者への提供（法第28条）の手続きは不要。

**反論の可能性**: 回答テキストの内容から間接的に個人が推定できるケース（非常に特徴的な回答、極少人数のクラス等）。

### Q2: Gemini API利用規約の18歳未満条項

規約: 「18歳未満の個人に向けた、またはアクセスされる可能性のあるサービスの一部として使用してはならない」

**主張**: 本サービスのエンドユーザーは教員のみ。生徒はアプリにアクセスしない。また、送信データは匿名化されており、「18歳未満の個人」のデータとして識別できない。

### Q3: 個人事業での提供は可能か？

教育機関向けに個人情報を扱うSaaSを、個人事業として提供する場合の法的リスク・要件。法人化の必要性。
