## 1. 短答問題（漢字・語句）

### 1.1 出題形式と実例

#### 漢字の読み
- 文中の下線部の読みをひらがなで書く形式
- 例: 「眺める」（ながめる）、「喝采」（かっさい）、「渓谷」（けいこく）
- 都立高校入試: 漢検準2級〜2級レベルが中心
- 共通テスト: 選択式。同じ漢字を含む語を五択から選ぶ（例:「ケイサイ」→「掲」を含む選択肢）

#### 漢字の書き取り
- 「下線部のカタカナを漢字に直しなさい」形式
- **小学校配当漢字のみ**から出題されるのが一般的（都立では過去12年間で中学配当漢字ゼロ）
- 例: 「荷物をトドける」→ 届ける、「実験を成功へとミチビいた」→ 導いた
- 配点: 読み・書き合わせて8〜10問、20点前後（東京都立の場合1問2点）

#### 語句の意味
- 四字熟語の意味選択（自業自得、弱肉強食、一期一会など）
- 慣用句の意味（「足を引っ張る」「顔が広い」）
- 同音異義語の使い分け（「計る」vs「測る」）
- 対義語・類義語を漢字二字で答える
- 熟語の構成（対義語型か類義語型か）

### 1.2 採点基準

#### とめ・はね・はらいの扱い
- **文化庁指針（平成28年）**: 「字体が読み取れ、その文字と判別できれば誤りとしない」
- **文科省見解**: 作文・読解では柔軟に、漢字テスト（書写能力を問う場合）では厳密に
- **漢検基準**: 字の骨組みが読み取れれば不正解としない。ただし画数不足・突き出し位置の誤り・類似別字との混同は不正解

#### 部分点
- 漢字の読み書き問題では**部分点なし**（正解 or 不正解の二択）
- 送り仮名の誤りは不正解

#### 旧字体・異体字
- 文科省: 「旧字体で回答しても誤りとしない」「許容範囲を認めていくことが大切」
- 漢検（2〜10級）: 旧字体は**不正解**

### 1.3 典型的な誤答パターン

#### 似た漢字の混同
- 部首の混同: しめすへん（礻）⇔ ころもへん（衤）、さんずい（氵）⇔ にすい（冫）
- 形の似た漢字: 織/識、構/講、租/祖、目/日、自/白、未/末
- 同音異義語: 計る/測る/量る/図る/謀る、努める/務める/勤める

#### OCRで問題になりやすい文字
- 横棒の本数: 三→二、土→士
- 全体の形が似た漢字: 鳥/烏、間/問、人/入、千/干、体/休
- 部首が似た漢字: 社/杜、待/侍、猫/描、酒/洒、候/侯
- 薄い筆跡・崩し字・続け字は認識精度が大幅低下
- 推奨スキャン解像度: 300dpi以上

## 2. 文法・語句問題

### 2.1 出題形式と実例

#### 品詞の識別（最頻出）
- 「下線部の品詞名を答えよ」: 「さわやかだ」→ 形容動詞、「大きな」→ 連体詞（形容詞「大きい」と混同しやすい）
- 「同じはたらきの語を選べ」: 「ない」の識別（形容詞 vs 助動詞 — 「ぬ」に置き換えで判別）
- 静岡県: 過去21年中7回出題、正答は全て副詞か形容詞

#### 敬語
- 穴埋め: 適切な敬語表現を選ぶ
- 書き換え: 普通語→尊敬語・謙譲語に直す
- 正誤判定: 敬語の誤りを指摘

#### 接続詞・助詞の穴埋め
- 「試験勉強を頑張りました。（　）、結果は良くありませんでした。」→ しかし
- 助詞・助動詞の識別: 「れる・られる」（受身/尊敬/自発/可能）、「ようだ」（比喩/推定/例示）

#### 慣用句・ことわざ
- 意味の選択: 「匙を投げる」「胸をなで下ろす」「後の祭り」
- 穴埋め: 体の部位や動詞を答える

### 2.2 採点基準

- **選択式**: 正誤のみ、部分点なし
- **記述式**: 部分点あり。おおむね20字に1要素で加点
- **表記の揺れ**: 統一基準なし。埼玉県「漢字で書けるところをひらがなで書いた場合、各学校の裁量で正答と認めて差し支えない」。ただし書き抜き問題は本文通りの表記が必要

### 2.3 典型的な誤答パターン

#### 品詞の混同
- 「大きな」を形容詞と回答 → 正しくは連体詞
- 「ない」を一律に助動詞と判断 → 「時間がない」は形容詞
- 「活用形」と「活用の種類」の混同

#### 敬語の混同（最多の誤り）
- 尊敬語と謙譲語の取り違え: 「先生に伺ってください」（誤: 伺うは謙譲語）
- 二重敬語: 「お帰りになられる」「おっしゃられる」
- 自分への尊敬語誤用: 「私のお考えを発表します」

#### 慣用句の誤用（入試で狙われるもの）

| 表現 | 正しい意味 | よくある誤解 |
|------|-----------|------------|
| 情けは人のためならず | 巡り巡って自分に返る | その人のためにならない |
| 役不足 | 役が軽すぎる | 能力が足りない |
| 敷居が高い | 義理を欠いて行きにくい | 高級すぎて入りにくい |
| 確信犯 | 信念に基づく行為 | 悪いと知りつつ行う |
| 気が置けない | 気を使わない間柄 | 油断できない |
| 煮詰まる | 結論間近 | 行き詰まる |
| 失笑 | 思わず笑う | 呆れて笑えない |
| 姑息 | その場しのぎ | 卑怯 |

## 3. 読解記述問題（短め）

### 3.1 出題タイプ

| タイプ | 設問形式 | 文末の型 |
|--------|---------|---------|
| 内容説明型 | 「〜とはどういうことか、説明しなさい」 | 〜こと。 |
| 理由説明型 | 「〜の理由を答えなさい」「なぜか」 | 〜から。/〜ため。 |
| 条件付き記述型 | 「本文中の言葉を使って〜字以内で」 | 設問に準拠 |
| 心情説明型 | 「どのような気持ちか」 | 〜気持ち。 |
| 書き抜き型 | 「本文中から○字で抜き出せ」 | 本文そのまま |

### 3.2 具体例と採点

**例: 麻布中（10点満点）** — 牛が「うんざりした」理由の記述
- 背景（秘密があること）: 3点
- 出来事（馬鹿にされた）: 3点
- 気持ち（うんざり・憂鬱）: 4点
- 満点解答: 「秘密を狸に知られ馬鹿にされたため、気が滅入っている」

### 3.3 採点の共通ルール

#### 要素方式（主流）
- **20字に1要素**が目安。60字指定→3要素、80字指定→4要素
- 各要素に配点を設定し、書けていれば加点
- 模範解答と完全一致不要。趣旨が同じなら認める（「馬鹿にされた」≒「笑われた」）

#### 内容の4分類
| 分類 | 説明 | 採点への影響 |
|------|------|------------|
| 必要的内容 | 答案の骨格 | 欠けると0点 |
| 有益的内容 | 主語明示・指示語解消 | 書くと加点、欠くと減点 |
| 無益的内容 | 問題文の繰り返し等 | 加減点なし（字数圧迫リスク） |
| 有害的内容 | 事実誤認・本文と矛盾 | 書くと減点 or 0点 |

#### 形式面の減点
| 項目 | 減点 |
|------|------|
| 文末ミス（「なぜか」に「〜こと」） | -1点 |
| 誤字脱字 | 1字ごとに-1点 |
| 句点なし | -1点 |
| 字数オーバー | 原則0点 |

### 3.4 典型的な解答パターン

**満点解答**: 全要素網羅、文末対応、字数8割以上使用
**部分点（要素不足）**: 「狸が笑ってきたので、怒っている」→ 3点/10点（背景欠落、気持ちの方向もズレ）
**部分点（気持ちの表現ズレ）**: 「秘密を狸に知られ馬鹿にされたので、許せない」→ 6点/10点
**0点解答**: 白紙、的外れ、本文の丸写し、字数超過

### 3.5 よくある失敗パターン
- 主語がない（「悲しかったから」— 誰が？何が？不明）
- 因果関係の省略（「本文にあるから省略」→ 採点者は内容のみで判断）
- 内容の重複（同じことを繰り返し、別の要素のスペースがなくなる）
- 設問条件の無視（「本文中の言葉を用いて」に自分の言葉だけ）
- 抽象的すぎる（「いろいろあったから」）

### 3.6 全国学力テストの実態
- 記述式の正答率: **25.6%**（選択式の6〜7割に比べ著しく低い）
- 「自分の考えとその理由を書く問題」の正答率: **17.4%**
- 無解答率: 約**30%**（27.7%）

## 4. 読解記述問題（長め・論述）

### 4.1 出題形式

#### 都立高校入試: 200字作文（10点満点）
- 論説文を読み、筆者の主張を踏まえて自分の考えと体験を200字以内で記述

#### 大阪府公立高校入試: 課題作文
- 資料を読み取り、条件に沿って意見を記述
- A問題12点（180字）、B問題18点（260字）、C問題20点（300字）

#### 共通テスト試行調査（参考: 最終的に導入見送り）
- 小問3題: 25字、50字、80〜120字
- 「確かに…しかし」構成で根拠と反対意見を記述

### 4.2 採点基準の詳細

#### 都立高校200字作文の加点・減点

**加点（10点満点）:**
| 項目 | 配点 |
|------|------|
| 自分の意見・主張がある | +4点 |
| 筆者の主張を踏まえている | +3点 |
| 具体的な体験や見聞がある | +3点 |

**減点:**
| 項目 | 減点 |
|------|------|
| 本文の抜き出し・要約のみ | 0点（不可） |
| 100字以下 or 201字以上 | 0点（不可） |
| 101〜150字 | -2点 |
| 論旨に一貫性がない | -2点 |
| 誤字・脱字1つ | -1点 |
| 誤字・脱字2つ以上 | -2点 |
| 文体の不一致（常体/敬体） | -1〜2点 |

#### 大阪府の配点構造
- 内容: 約7割（主張の明確さ、例示の的確さ、論理性）
- 表記: 約3割（誤字脱字、文法、原稿用紙の使い方）

#### JASSO日本留学試験の段階評価（参考）
| 得点 | レベル | 基準 |
|------|--------|------|
| 50点 | S | 課題に沿い、説得力ある根拠と効果的な構成 |
| 45点 | A | 課題に沿い、妥当な根拠と適切な表現 |
| 40-35点 | B | おおむね妥当、情報伝達上の支障なし |
| 30-25点 | C | 根拠の妥当性・構成・表現に不適切な点あり |
| 20-10点 | D | 主張や構成の欠如、表現に重大な問題 |
| 0点 | NA | 白紙、無関係、40字未満 |

### 4.3 「惜しい」解答 vs 「的外れ」解答

**「惜しい」解答:**
- 方向性は正しいが1要素が欠けている
- 感情の方向性（プラス/マイナス）は合っているが表現がズレ
- 文法的誤りはあるが内容は読み取れる

**「的外れ」解答:**
- 問われていることを誤解（理由を聞かれているのに内容説明）
- 本文と無関係な自分の感想
- 傍線部と異なる場面の内容
- 設問条件をそもそも読み落としている

## 5. 小論文・作文

### 5.1 出題形式

#### テーマ型（最多）
- 「将来の夢と、その実現のための高校生活」（600〜800字）
- 「新しいことに挑戦することの大切さ」（550〜600字）
- 頻出: 中学校生活で頑張ったこと、高校での抱負、SDGs、AI・情報化

#### 資料読解型
- グラフ・統計データを踏まえて論述
- 例: 食品ロスに関する資料分析（500字以内）

#### 課題文型
- 提示された文章の筆者の主張を踏まえて意見を述べる
- 大学入試: 300〜2000字の課題文 → 要約＋自分の意見

#### 字数の傾向
- 都立高校（国語内作文）: 200字以内
- 公立高校（推薦・前期）: 400〜800字
- 大阪府: A問題180字、B問題260字、C問題300字
- 大学入試小論文: 600〜800字が標準
- **共通ルール: 指定字数の8割以上が必須**

### 5.2 採点基準（4つの評価観点）

優先順位順:
1. **設問への適合性**（最重要）: 問われていることに正確に答えているか
2. **論旨の一貫性**: 議論が整理され、一貫しているか
3. **内容の説得力**: 根拠に基づいた踏み込んだ内容か
4. **文章力・表現力**: 誤字脱字なし、文体統一、適切な表現

#### 原稿用紙の減点項目

| ルール | 正しい使い方 | よくあるミス |
|--------|-----------|-----------|
| 段落の書き出し | 1マス空ける | 空けない |
| 行末の句読点 | 最後のマスに文字と共に | 行頭に句読点 |
| 閉じかっこ＋句点 | 1マスにまとめる（。」） | 別々のマス |
| 縦書きの数字 | 漢数字 | アラビア数字 |
| 文体 | 「だ・である」統一 | 「です・ます」混在 |

### 5.3 典型的な解答パターン

#### 高評価の特徴
- 序論で結論を明示
- 根拠を2つ以上挙げ、各々に具体的エピソード
- 序論→根拠1＋具体例→根拠2＋具体例→結論の明確な構成
- 文体統一、字数を適切に使い切り

#### よくある失敗パターン

| 失敗パターン | 影響 |
|-------------|------|
| 設問に答えていない（受験者の5割以上） | 最大の減点、0点もあり |
| 主張が不明確（設問の要約に終始） | 大幅減点 |
| 根拠がない・主観的（「〜だと思います」だけ） | 説得力不足 |
| 体験談だけ（考察がない） | 内容点が低い |
| 話題の散らばり（焦点がぼやける） | 構成力の評価低下 |
| 字数不足（8割未満） | -2〜-10点 |
| 字数超過 | 都立: 0点 |
| 文体の混在 | 表記減点 |
| 一文が長すぎる（60字以上） | 主述崩壊 |
| 条件の見落とし（「体験を交えて」等の指示無視） | 加点されず |

## 6. 横断的知見

### 6.1 採点方式の使い分け
- **読解記述**: 要素ごとの**加点方式**が主流
- **作文**: 満点から出発する**減点方式**が多い
- **表記面**: 常に**減点方式**（誤字脱字、文末ミス等）

### 6.2 AI採点で特に注意すべきポイント
1. **文末の型と設問の対応**: 「なぜか」→「〜から」、「どういうことか」→「〜こと」
2. **要素の同義判定**: 「馬鹿にされた」≒「笑われた」を同一要素と認める柔軟さ
3. **感情の方向性**: プラス/マイナスの方向が合っていれば、細かな表現差は許容
4. **字数制限の厳格性**: 超過は原則0点、不足は段階的減点
5. **漢字の表記揺れ**: ひらがな書きの許容度は問題タイプ・学校によって異なる
6. **OCRノイズの影響**: 似た漢字の誤認識を前提とした前処理が必要

### 6.3 問題タイプ別の採点難易度

| タイプ | AI採点の難易度 | 理由 |
|--------|--------------|------|
| 漢字読み | 低 | 完全一致判定で十分 |
| 漢字書き | 中（OCR依存） | 手書き認識精度に左右 |
| 文法・選択式 | 低 | 記号選択 |
| 読解記述（短め） | 中〜高 | 要素の同義判定が必要 |
| 読解記述（長め） | 高 | 論理構成の評価が必要 |
| 小論文・作文 | 最高 | 内容・構成・表現の総合評価 |

## 7. 東京大学の国語

### 7.1 試験構成

| 項目 | 文科 | 理科 |
|------|------|------|
| 試験時間 | 150分 | 100分 |
| 満点 | 120点 | 80点 |
| 大問数 | 4題 | 3題 |

- 第一問: 現代文（評論）— 文科・理科共通
- 第二問: 古文 — 文科・理科共通（文科は設問数多い）
- 第三問: 漢文 — 文科・理科共通（文科は設問数多い）
- 第四問: 現代文（随筆/小説）— 文科のみ

**予備校推定の配点**: 第一問40点、第二問30点(文科)/20点(理科)、第三問30点(文科)/20点(理科)、第四問20点(文科のみ)

### 7.2 解答欄の特徴

- **字数制限なし、解答欄の大きさで暗黙的に制限**
- 1行の長さ: 約14cm（約30〜35字）
- 通常の設問: 1行（30字前後）or 2行（60字前後）
- 第一問の最終設問: 「100字以上120字以内」と字数指定ありでマス目提供

### 7.3 設問パターン（3類型のみ）

漢字の書き取りを除き、東大現代文の設問は以下の**3パターンしかない**:

1. **「どういうことか、説明せよ」** — 傍線部の内容説明。抽象・比喩を具体的に言い換え。文末「〜こと。」
2. **「それはなぜか、説明せよ」** — 理由説明。因果関係を説明。文末「〜から。」
3. **「なぜそういえるのか、説明せよ」** — 根拠説明。等価＋因果の性質を持つ。文末「〜から。」

### 7.4 採点の実態

#### 東大の公式姿勢
- 模範解答・出題意図・採点基準を**一切公表しない**
- 「解答は1つでないことがいっぱいある」「解答を導くまでのプロセスを見たい」（福田裕穂理事・副学長, 2018年）
- 「1つだけが解答と捉えられてしまうことを非常に恐れている」

#### 予備校模試 vs 東大本試験の根本的な違い
- **予備校模試**: 要素加点方式（キーワードがあれば加点、なければ0点）。大学生バイトがマニュアル通り採点
- **東大本試験**: **解答全体の方向性・論理的一貫性を重視する総合評価**。大学教員が採点し、答案全体としての理解度を見る
- 得点開示データの分析から、模試の要素加点では説明できない得点分布が確認されている

#### 近年の得点データ
- 2024年度: 文科合格者平均64.0/120（53%）、理科合格者平均41.5/80（52%）
- 文科の合否差は約9点。目標は文系60〜70点、理系40点前後

### 7.5 具体的な出題例（2024〜2026年度）

**2024年度第一問**: 小川さやか「時間を与えあう」
- 「行商人たちにとって掛け売りを認めることは、商売戦略上の合理性とも合致していた」—なぜか、説明せよ（2行）

**2025年度第四問**: 佐多稲子「狭い庭」— **21世紀初の小説出題**

**2026年度第一問**: 松本卓也『斜め論 空間の病理学』
- 設問四: 「ふたつの対話の協同」について本文全体の趣旨を踏まえて説明（120字）

**2026年度第三問**: 白居易『白氏文集』— **近年にない漢詩出題**

### 7.6 古文・漢文の採点ポイント
- **古文**: 直訳・逐語訳が基本。主語の明示・敬語の訳出が特に重い。「主語を明らかにして」等の条件を満たさないと正解にならない
- **漢文**: 語句の意味、現代語訳、対句構造の理解。書き下し文を経由した訳出

### 7.7 高得点のコツと典型的な減点パターン

**高得点のコツ:**
- 設問文の徹底精読（出題者の意図把握が最優先）
- 「問い」に正面から答える（拙くても方向性が合っていれば得点）
- 抽象と具体の対応関係の把握
- 意味段落ごとに傍線が1つ → 解答範囲の判断材料
- 120字問題は「要約」ではなく傍線部の説明問題として解く

**典型的な減点:**
- 設問の意図とズレた解答（方向性の誤りは致命的）
- 本文のフレーズの無思慮な寄せ集め（「ツギハギ」は評価されない）
- 日本語として不自然な文章（採点者は東大教授＝国語のエキスパート）
- 古文の条件付き訳で条件無視
- 古文の主語欠落（配点が他の文法要素より重い）

## 8. 京都大学の国語

### 8.1 試験構成

| 項目 | 文系 | 理系 |
|------|------|------|
| 試験時間 | 120分 | 90分 |
| 満点 | 150点 | 100点 |
| 大問 | 3題（現代文2＋古文1） | 3題（現代文2＋古文1） |

- 漢文は独立した大問として出題されない（東大との大きな違い）
- 理系は文系と共通問題だが、各1問ずつ設問が少ない

### 8.2 解答欄の特徴
- 横1cm × 縦14cmの行が複数並ぶ形式
- 設問ごとに2〜5行
- 1行あたり20〜25字 → 1問あたり約40〜125字
- 現代文全体で小問8〜10題、**総記述量は約1000字**

### 8.3 東大との決定的な違い

| 観点 | 東大 | 京大 |
|------|------|------|
| 解答欄 | 小さい（1〜2行） | 大きい（2〜5行） |
| 求められる力 | 本文の要素を的確に抽出し簡潔にまとめる | 本文を自分の中に取り込み、自分の言葉で再構成する |
| 解答方略 | 本文のキーワードをつなげば形になる | キーワードのつぎはぎでは体をなさない |

京大の出題方針（公式）: 「日本語の文章の論理や論旨を的確に理解し、**自らの言葉で論理的にその理解を表現できること**を評価します」

### 8.4 採点基準
- 京大は公式模範解答を公表する方針
- 古文: 助詞・助動詞の訳出に個別配点（例: 3点、6点）
- 部分点がつきやすい傾向。方向性が合っていれば一定の得点が期待できる
- 差がつくポイント: 助詞・助動詞の細部、文脈を踏まえた言葉の補足、設問要求を満たした自然な文

## 9. 一橋大学・旧帝大の国語

### 9.1 一橋大学
- **100分、全問記述式、大問3題**
- 第3問の**200字要約**が最大の特徴（3000〜4000字の評論を200字以内に圧縮）
- 要約の採点観点: 必要要素の網羅性、要素間のつながり、主述のねじれ・省略・係り受けの不備
- 記述量は全体で約700字。少ない字数に凝縮する圧縮力が鍵

### 9.2 旧帝大の横断比較

| 大学 | 記述の特徴 | 1問の字数目安 | 難易度 |
|------|-----------|-------------|--------|
| 東北大 | 字数制限が極めて厳しい | 30〜60字 | やや難 |
| 名古屋大 | 記述中心、漢文150字要約が定番 | 可変 | 標準〜やや難 |
| 大阪大 | 文学部は字数制限なしで記述量膨大 | 可変 | 難関 |
| 九州大 | 字数制限なし多め、解答量多い | 可変 | 標準〜やや難 |
| 北海道大 | 字数制限付き多め | 20〜90字 | 標準 |

### 9.3 各大学が求める方向性
- **東北大**: パラフレーズ力。極めて短い字数に要素を凝縮する表現力
- **名古屋大**: 基礎的な読解・記述力の正確さ
- **大阪大**: 文学部は踏み込んだ読解力。和歌の設問が頻出
- **九州大**: 抽象的表現の説明が頻出。古文で文学史が毎年出題
- **北海道大**: 標準的な読解力。100字以上の論述が例年1題

### 9.4 国立大入試に共通する採点の5観点
1. **要素の網羅性**: 設問が求める要素が漏れなく含まれているか
2. **論理の一貫性**: 要素のつなぎ合わせが論理的に通っているか（「ツギハギ」は見抜かれる）
3. **日本語の正確さ**: 主述のねじれ、係り受けの不備、不自然な省略がないか
4. **設問への対応**: 換言/理由/根拠に正確に応えているか
5. **文脈の把握**: 傍線部前後だけでなく文章全体の趣旨を踏まえているか

## 10. 予備校模範解答の信頼性と批判的検討

### 10.1 予備校間の解答の相違

#### UmeeTの「東大現代文解答コンテスト」
- 東大生12名が5大予備校の模範解答を採点
- **最高88点 vs 最低58点**（30点差）
- 58点の予備校は東大合格ライン（60%）を下回る

#### 予備校ごとの傾向
- **赤本 = 河合塾 = 東進**: オーソドックス
- **駿台**: 「換言が大胆すぎて本番では怖くて書けない」独特の解答
- **代ゼミ**: 近年質が低下傾向との指摘

#### 具体的な相違事例
- 2016年度東大第一問: 「さしあたり…代えることができる人」を「喜びとする人」と着地させた予備校（本文に「喜び」の根拠なし）
- **素材文の著者（明治大学の重田園江教授）が、早稲田大学の発表した解答と自分の解答が違うと問い合わせた**事例
- 赤本にも「大学生バイトが解いている」ケースがあり、「間違い以外にもセンスのない答え」が載ることがある

### 10.2 予備校解答の構造的問題

1. **速報解答の質的限界**: 数時間で少人数が作成。十分な吟味なし
2. **本文にない内容の付加**: 本文で言及されていない要素（「喜び」「内省」等）を勝手に導入
3. **模範解答が完璧すぎる**: 受験生には書けないレベルで、記述への苦手意識を助長
4. **キーワード至上主義の弊害**: 「キーワードがあれば加点」方式が、意味を理解せずツギハギする学習習慣を生む

### 10.3 本番の採点の実態（元出題者・元採点者の証言）

- **協議型採点**: 「新しい間違い方が見つかるたびに採点グループ全員で相談し、評価を検討する」
- **匿名採点**: 受験番号すら分からない状態で評価
- **採点基準の動的洗練**: 事前に基準を設けつつ、実際の答案を見ながら基準を洗練させる
- **日本史の出題意図公表例**: 「重点の置き方によっては朝鮮戦争に触れない解答もありえる」→ 採点基準に幅があることを大学が認めた
- **書かれていることだけが受験生の考え**: 「答案に書かれていない行間を補って点を出す」ことは禁止

### 10.4 定期テストの作問品質の問題

- 初めてのテスト作成で「4時間で10点分しかできなかった」→ 全体で30時間以上かかるケースも
- **ありがちな問題**: 誤植、テスト範囲外からの出題、正解がない問題、問題文の曖昧さ
- 「授業中に説明したキーワードが入っていない」で不正解にする事例（本文のみで判断すべき原則に反する）
- 教員不足で専門外教員が代行、作問研修の不足、多忙化が背景

### 10.5 批判的検討の方法論（AI採点チューニングへの示唆）

#### 模範解答の妥当性検証の6観点
1. **本文根拠**: 解答の内容が本文中に根拠を持つか
2. **論理的整合性**: 主述の対応、因果関係・並列関係が適切か
3. **設問対応**: 設問が求めることに過不足なく答えているか
4. **過剰解釈**: 本文に書かれていない意味を読み込んでいないか
5. **表現の妥当性**: 大胆すぎる換言やツギハギがないか
6. **複数解答の比較**: 共通要素と相違点を分析し、最も妥当な解を再構成

#### 「正しい解答」の性質
- 数学のような一意的な正解は存在しない
- 採点基準となる要素を含み、本文の趣旨を正しく捉えていること
- 趣旨が合っていれば表現が異なっても認められる（一定の幅がある）
- 客観的根拠は基本的に本文中の語句・表現に求められる

#### 共通テスト記述式導入見送りの教訓
- 50万人の答案を20日で採点する計画 → 採点のぶれを防げず断念
- 自己採点と実際の採点の一致率: 国語で**66.0〜70.7%**
- 「採点ミスを完全になくすことは難しい」と文科省が最終判断
